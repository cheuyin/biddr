## Tech Stack
- Frontend: 
    - React
    - JavaScript
    - ChakraUI
- Backend:
    - Node.js
    - Express.js
    - PostgreSQL

## Starting the Server
1. At the project root, type `cd backend`
2. Run `npm install`
3. Start the server with `npm run dev`
4. Go to http://localhost:3000/
